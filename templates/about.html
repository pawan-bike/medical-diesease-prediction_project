Medical Disease Prediction System

This project is a Machine Learning based Medical Disease Prediction System. It predicts diseases based on the patient's health information entered by the user. The prediction is generated using a trained machine learning model.

Input Features

  • Age
  • Gender
  • BMI
  • Blood Pressure
  • Glucose Level
  • Cholesterol
  • Heart Rate
  • Smoking
  • Alcohol
  • Physical Activity
  • Family History

Prediction

The Machine Learning model analyzes all the above health parameters and predicts the patient's Disease.

Technologies Used

  • Python
  • Flask
  • Pandas
  • Scikit-Learn
  • HTML
  • CSS
